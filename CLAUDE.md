# CLAUDE.md

이 파일은 Claude(에이전트)가 이 저장소에서 작업할 때 따라야 할 지침을 모아둔 곳입니다.
사용자가 직접 검토·수정하기 위한 문서이며, 새 세션이 시작될 때마다 자동으로 읽힙니다.

> 프로젝트: **Connect AI v2 (P-Reinforce)** — VS Code / Cursor 확장 프로그램.
> 사용 언어: 사용자와의 대화는 기본적으로 **한국어**로 진행합니다.

---

## 🎬 영상 제작 (Video) — Remotion 사용

사용자가 **영상 제작 / 동영상 / 비디오 / 모션 그래픽 / 애니메이션 영상**을 요청하면
**반드시 [Remotion](https://www.remotion.dev/)** 을 사용해 React 코드로 제작한다.

- 이 저장소에는 **`remotion-best-practices` 스킬**이 설치되어 있다
  (`.claude/skills/remotion-best-practices` → `.agents/skills/...`).
  Remotion 코드를 다룰 때는 **먼저 이 스킬을 참고**한다 (자막, 오디오, 전환,
  텍스트 애니메이션, ffmpeg, 폰트, 차트 등 주제별 규칙이 `rules/`에 정리되어 있음).
- 빈 폴더/새 프로젝트라면 다음으로 스캐폴딩한다:
  ```bash
  npx create-video@latest --yes --blank --no-tailwind <project-name>
  ```
- 애니메이션은 `useCurrentFrame()` + `interpolate()` (필요 시 `Easing`)로 구현한다.
- 다른 영상 라이브러리(예: ffmpeg 단독 합성, moviepy 등)는
  **사용자가 명시적으로 요청한 경우에만** 사용한다.

## 🧩 새 스킬 / 새 서비스 접근 권한 기록

새로운 스킬이나 외부 서비스(MCP 서버, API, CLI 등) 사용 권한을 받게 되면,
**아래 표에 그 내용을 정리**한다. (무엇을 / 언제 / 어떻게 쓰는지를 남겨,
다음 세션에서도 같은 맥락으로 활용할 수 있도록 한다.)

기록 규칙:
- 권한을 새로 받으면 **즉시 이 표에 한 줄 추가**한다.
- 설치형 스킬은 설치 명령과 설치 위치(프로젝트 로컬 / 글로벌)를 함께 적는다.
- 영속성 주의: 이 실행 환경은 **컨테이너가 매번 초기화**되므로,
  유지하려면 반드시 **git에 커밋**해야 한다. (프로젝트에 커밋된 것만 새 세션에서 살아남음)

### 설치된 스킬

| 스킬 | 용도 | 설치 방법 | 설치 위치 | 추가일 |

## ⚙️ 작업 일반 규칙

- 변경 사항은 지정된 **개발 브랜치**에 커밋·푸시한다 (임의로 `main`에 직접 푸시 금지).
- 푸시 후에는 **draft PR**을 생성한다(없을 경우).
- 되돌리기 어려운 작업(머지, 삭제, 외부 전송 등)은 **먼저 사용자에게 확인**한다.
- 스킬/규칙 파일은 에이전트 전체 권한으로 실행되므로, 새로 설치한 스킬은
  사용 전 한 번 내용을 검토한다.
